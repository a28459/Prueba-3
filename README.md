## Descripcion del Torneo Videojuegos
Web para un torneo de videojuegos creada para el ayuntamiento. Es una pagina estatica hecha con Bootstrap 5 para informar de horarios, sitio y registro.

## Secciones y Grid
- Nav: Barra de Navegación.
- Hero: Cabecera con fondo oscuro y titulo del evento.
- Info: Grid de 2 columnas md-6 para texto e imagen.
- Horario: Grid de 3 columna md-4 usando tarjetas de Bootstrap.
- Ubicacion: Un div centrado imagen de la ubicacion de fondo.
- Registro: Seccion oscura con un boton grande btn-lg.
- Footer: Datos de la empresa y copyright.

## Componentes de Bootstrap usados
- Navbar (navbar-dark)
- Cards (card-body)
- Buttons (btn-primary)
- Sistema de rejilla (container, row, col)
- Listas (list-group)
